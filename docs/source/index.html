    源代码

    “言”项目的代码被分为许多子项目，所有代码都托管在 Github 上。

    编译器框架

    • Yan Foundation : 面向老师和学生的编译器框架。
    • Yan Language : 为阐述Yan Foundation设计的一款微型语言。
    • Token util : 面向老师的词法单词类型代码生成工具。

    自动评测系统

    • OJ Website : 自动评测系统的网站前后端。
    • OJ Core: 自动评测系统的内核。

    实验框架

    • YCC : 基于Yan foundation的C语言编译器实现框架。
